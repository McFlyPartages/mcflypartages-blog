Retrouver le sudo dans Debian 10 et 11

Mis à Jour : 9 Janvier 2022
Temps de lecture 1 minute
linux
#bullseye, #buster, #debian

Comme vous avez dû le remarquer depuis la version 10 de Debian nommée Buster, une commande avec sudo renvoie que sudo est introuvable. Et oui depuis cette version, Debian n’installe plus sudo par défaut. Voici la méthode pour retrouver cette commande. Installation. Dans un terminal, connectez-vous en SSH Passer en root (si vous êtes connecté via un utilisateur) puis procéder à l’installation avec les commandes suivantes : su root apt install sudo -y
